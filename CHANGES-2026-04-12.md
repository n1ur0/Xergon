# Xergon Network - Changes Summary (April 12, 2026)

## 🎯 Major Updates

### Security Fixes (CRITICAL)
- **SEC-001**: Fixed authentication bypass in `xergon-relay/src/handlers.rs`
  - Removed default API key fallback
  - Now requires explicit `X-API-Key` header
  - Returns 401 for missing/invalid keys

- **SEC-002**: Fixed timing attack vulnerability in `xergon-relay/src/auth.rs`
  - Implemented `const_time_eq()` function
  - Prevents signature timing analysis
  - Constant-time comparison

- **SEC-003**: Fixed hardcoded cookie placeholder in `xergon-marketplace/lib/stores/auth.ts`
  - Changed `AUTH_COOKIE = "***"` to `"xergon_auth"`
  - Proper cookie naming
