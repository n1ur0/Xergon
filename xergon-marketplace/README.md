# Xergon Marketplace

Web marketplace for discovering and using AI models on Xergon.

## Stack

- Next.js 15 + TypeScript + Tailwind CSS v4

## Quick Start

```bash
npm install
npm run dev
```

## Configuration

Create `.env.local` with:

```
NEXT_PUBLIC_API_BASE=http://127.0.0.1:8080
```
