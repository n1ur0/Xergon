# Xergon Relay

API relay server for the Xergon decentralized AI compute network.

## Stack

- Rust + Axum + SQLite + Tokio

## Quick Start

```bash
cp .env.example .env
cargo run --release
```

## Configuration

Set environment variables (see `.env.example`) or use defaults for development.
Server runs on port 8080 by default.
